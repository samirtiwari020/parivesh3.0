🌿 PARIVESH 3.0
AI-Powered Environmental Clearance Workflow System

PARIVESH 3.0 is a digital platform that modernizes the **Environmental Clearance (EC) process** using AI-driven document analysis and role-based governance workflows.
It helps applicants, government authorities, and expert committees manage environmental proposals more efficiently while reducing manual document review.

---

🚀 Problem

Environmental clearance proposals contain **hundreds of pages of documents**, making manual review slow and inefficient.

---

💡 Solution

PARIVESH 3.0 uses **AI-based GIST generation** to automatically analyze uploaded PDFs and produce structured proposal summaries, enabling faster decision-making for review committees.

---

⚡ Key Features

Applicant Portal
Submit proposals, upload documents, and track application status.

State Authority Dashboard
Verify proposals, raise **EDS (document shortfall)** queries, and forward applications.

Central Authority Review
Advanced verification and approval pipeline management.

AI GIST Generator
Automatically extracts key insights from proposal documents.

Committee Review System
Review AI summaries, record discussions, and generate **Minutes of Meeting (MoM)**.

GIS Map Visualization
Map-based project tracking with status indicators:
Green – Approved | Yellow – Under Review | Red – Rejected

---

🛠 Tech Stack

Frontend: React.js, Leaflet
Backend: Node.js, Express.js
Database: MongoDB
AI: Gemini API
Security: JWT, bcrypt
File Handling: Multer, pdf-parse, PDFKit
